   https://github.com/the-hideout/tarkov-dev-svg-maps

   Licensed under Creative Commons Attribution-NonCommercial-ShareAlike 4.0
   International (CC BY-NC-SA 4.0).
   https://creativecommons.org/licenses/by-nc-sa/4.0/

   The SVG files are bundled byte-for-byte as published. The app rewrites a
   copy's viewBox in memory at display time to orient the map; the distributed
   files are unmodified.

   That project forbids use of its assets in software that facilitates cheating
   (in-game radar/ESP overlays, cheat clients, pixel-bots). This app reads no
   game memory, renders nothing in-game, and has no knowledge of live raids --
   it plots public quest data on a static map.


2b. The Labyrinth map tiles by tarkov.dev  --  maps/Labyrinth.svg
   https://tarkov.dev

   No SVG of The Labyrinth exists in the SVG maps project. The bundled image
   is tarkov.dev's own top-down tile render of the location (the imagery
   their interactive Labyrinth page serves; maps.json lists "Tarkov.dev" as
   its author), stitched from their published tile pyramid, cropped to
   content and wrapped in an SVG shell so the app can treat it like its
   other maps.

   The render is derived from Escape from Tarkov's game art, which belongs
   to Battlestate Games -- the same footing as the trader portraits and
   location screenshots in section 5: identification in a free,
   non-commercial fan tool. If tarkov.dev, the-hideout or Battlestate Games
   would prefer it not ship this imagery, it comes out on request.
